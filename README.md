 # Sales & Operations Performance Dashboard

## Overview
This project focuses on analyzing sales and operational performance using SQL and Tableau.
The analysis highlights key trends, regional performance, and product-level insights.

## Tools Used
- SQL (SQLite)
- Tableau Public
- Superstore Dataset

## Key KPIs
- Total Sales
- Total Profit
- Monthly Sales Trend
- Regional Performance
- Product-Level Analysis

## Tableau Dashboard
Live Dashboard Link:
https://public.tableau.com/

sql/queries.sql – SQL queries used for analysis
